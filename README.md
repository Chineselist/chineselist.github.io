# Chineselist
[Chineselist](https://groups.google.com/g/chineselist) 是在芬华人的 **开放** 信息发布平台，用来 **补充** 微信群和公众号信息发布的 **平台限制** 。

## 微信群限制

- 需要主动加入才能收到信息
- 上限500人
- 过多信息和被折叠群
- 无法跨群搜索

## 公众号限制

- 无搜索功能
- 作者无法发布网页链接

## Chineselist 的补充

1. 开放，内容无屏蔽
2. 发布信息简单迅速：网页+邮件
3. 跨所有信息内容搜索
4. 公益，非盈利
5. 支持邮件推送最新信息
